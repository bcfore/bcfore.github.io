Projects

Most of these projects were a part of my Launch School course work. The front-end assignments were usually of the form "Implement such-and-such functionality using JavaScript (or jQuery or Backbone.js)" or "Using HTML and CSS, reproduce such-and-such website." (I haven't included any walk-throughs.)

Some of the projects are deployed on Heroku. These might take a few seconds to get going after you open them.

  • My website (ie, this site)

    I've only recently (July 2017) set up this site, and I'm still working on it. I'm using GitHub Pages because they offer free, secure (https) hosting. It's given me the chance to learn Jekyll, Sass, and Liquid (templating).

    Here's a link to the repo.

  • Trello Clone

    This is my implementation of a Trello board, done as part of the final examination for Launch School's front-end curriculum. Basically, I was given a week and a half to write the HTML/CSS and then implement as many of the interactive features as I could. It uses Backbone.js as the MVC framework, with an Express server for the back-end.

    This is probably my most sophisticated front-end project to date.

    You can access the source code here.

  • Sushi site

    A single page app built using Backbone.js and jQuery on the front-end, with Express and Node.js on the back-end. We were given the HTML and CSS, and tasked with implementing the dynamic functionality.

    See the source code for more details.

  • Clone of JSConf BR 2014

    The task was to reproduce the layout and dynamic CSS effects of the JSConf BR 2014 site. (The images come from the official website, but the HTML and CSS is my own.)

    This was part of my exam preparation for the HTML/CSS portion of the Launch School curriculum. I find it useful to refer back to this from time to time, to remind myself of how to implement certain stylistic effects.

  • Nutrition Facts label

    The task here was to reproduce this image of a nutrition facts label (which comes from a Wikipedia article) without using any classes or ids to aid in the styling. Ie, we had to rely on CSS pseudo-classes and combinators. (The image I worked off of was zoomed in to about 150% -- that's why my version seems magnified.)

  • Positioning rules summary

    This is my clone of a page with several examples of how to position elements. It's useful as a quick refresher on the box model, floats, and positioning.

  • Simple Smashrun

    This is a Ruby/Sinatra app that I wrote as a practice project prior to an exam in Launch School's back-end curriculum. (I didn't know much HTML/CSS/JS at the time, hence the minimal UI.)

    In the source code, the main Ruby code is in bcf_smashrun.rb, and the Minitest suite of tests are in test/bcf_smashrun_test.rb.
